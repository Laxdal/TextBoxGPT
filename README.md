To use the script, you can copy your prompt to your clip board, then press Ctrl+D to activate the bot, or just add GPT infront of your prompt then copy them to the clip board, this will also wake the llm up. For example, you want to say: Help me enhance this email. You will need to add GPT at the front of it like this: GPT Help me enhance this email. Then the program will detect it and execute the prompt. 
You can also customize your api, proxy url, and the name of the model.
For the input part it uses a input dll from:
https://github.com/bukepro/buke_km_lib/tree/master by https://github.com/bukepro
to make sure this will work under all conditions.
